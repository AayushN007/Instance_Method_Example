````markdown
# Employee Class Program (Python)

## 📌 Description
This Python program demonstrates the use of **classes, objects, and constructors** in Object-Oriented Programming (OOP).

The program creates an `Employee` class with attributes like name, employee ID, salary, and department. Objects are created using the constructor and their details are accessed using object references.

## 🧠 Concepts Used
- Class
- Object
- Constructor (`__init__`)
- Instance Variables
- Object Attribute Access

## 💻 Code Explanation

The `Employee` class contains:
- `name` → Stores employee name
- `id` → Stores employee ID
- `salary` → Stores employee salary
- `dept` → Stores department name

Three employee objects are created:
- Employee 1: Arjun
- Employee 2: Priya
- Employee 3: Karan

## ▶️ How to Run

1. Install Python
2. Download or clone this repository
3. Run the Python file:

```bash
python Employee.py
````


This program is created for practicing basic Python OOP concepts and understanding how constructors initialize object data.
